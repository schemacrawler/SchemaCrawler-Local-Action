## About

The *SchemaCrawler (Local) Action for GitHub Actions* allows you to run a SchemaCrawler report of your database using a GitHub Actions workflow for your project. SchemaCrawler is installed locally on your Linux-based runner.

See [an example GitHub Actions workflow](https://github.com/schemacrawler/SchemaCrawler-Action-Usage-Example) that shows how to use the SchemaCrawler Action for GitHub Actions.
